## 2. 模块 B：战斗反馈大升级 hotfix-y2 / y3 / y4 / y5

### 改造目标
旧：扣血瞬间触发 A 鱼震动 + B 飘字 + D 音效 + E 粒子（PHASE 21-1 D14 hotfix-o）
新：扣血变成**两阶段过场**

```
3 秒 tick 到点
  ↓
阶段 1：拉力条 ▲ 指针 → 3 颗光弹齐射 → 飞向鱼（300ms 抛物线弧）
  - 光弹三层结构：外圈呼吸光晕 + 主体 + 偏上高光
  - 飞行中每 30ms 喷 3 颗粒子火花（带重力下落）
  - 起点闪光 8 颗金粒子
  ↓
阶段 2：第 3 颗光弹命中鱼身瞬间
  - 真扣 HP（旧 _onFishHit 四件套：抖动 + 飘字 + 音效 + 命中粒子）
  - 飘字数字四舍五入到整数（避免 -5.288899999... 长尾）
```

### Combo 系统（方案 2 温和版）

## 3. 模块 C：死循环 / 体验 bug 修复

### 3.1 HP=0 出水演绎 latch（hotfix-x4）
**问题**：HP=0 后玩家松手会触发回血（hpRecoverPerSecond），fishHP 立刻 > 0 → fishExhausted 又变 false → 上拉中断 → 玩家又按住扣血到 0 → 又松手回血 → 死循环"持续拉扯"。

**修复**：用 `_fishExhaustedLatch` 锁存第一次 HP<=0 的状态，进入"出水演绎专属分支"：
- HP 锁死 0（防 UI 飘）
- tension 自动慢降 30/s（视觉自然松线）
- 鱼 Y 自动 -225/s 上拉
- 鱼 X 自动靠拢屏幕中央 90/s
- **所有 Failed 路径全部免疫**（爆条 / escape / 飘出屏幕 / 鱼线绷断 / 自然挣脱）
- 玩家任何输入都被忽略
- 到 catchZoneY → Caught + _onFishCaught

### 3.2 鱼群钓获减数（hotfix-z）
**问题**：鱼被钓走后鱼群里那条鱼仍在数组里，HUD 浮窗显示密度永不衰减。

**根因**：原方案试图用 `bobberFSM.follower` 引用删鱼，但 `_startBite` 进 BiteWindow 时已 `resetBobberApproach` 清空 follower。

**修复**：
- `fish-group-system` 新增 `takeHookedFishRef()` / `removeFishFromGroup(ref)`
- `_startBite` 在 reset 之前抓引用存到 `fishing-scene._hookedFishRef`
- `_onFishCaught` 用缓存引用真正删除（兼容鱼飘到其他群的兜底）
- 失败/跑鱼时主动清掉引用，鱼仍留在群里（合理）

### 3.3 钓鱼场景进入回 Idle + 回村按钮事件冲突（hotfix-y）
**问题**：钓鱼场景下点回村按钮 → mousedown 同帧触发 `_tryStartAimingByInput` → 切场景后 Aiming 残留 → 下次进来死循环。

**双重保险修复**：
1. **事件层防冲突**：`_mouseDownAimHandler` 先判定鼠标是否在回村按钮命中盒内，是则直接 return
2. **状态层强制重置**：`init()` 每次进场景强制 `fsm.state='Idle'` + 清空所有鼠标/键盘按住标记 + 清战斗残留 + 清前戏 FSM

### 3.4 鱼线终点回中心 + 弧度平滑（hotfix-v）
- **终点位置**：尝试过 0.42→0.47 的鱼嘴 ratio，最终因 10 种鱼形态差异太大（曲腰鱼下垂嘴 / 鲤鱼鱼须 / 朝向左右翻转）放弃，**回到鱼中心**（被鱼身遮挡，不穿帮，零维护）
- **弧度形状**：`bias` 0.75 → **0.5**（对称悬链线），`curveFactor` ×0.6（避免对称后过沉）
- **粗细**：lineWidth 4 → 2（细一倍）
- **钩点缩小**：8×8+4×4 → 4×4（够小，靠鱼身遮挡）

### 3.5 拉力条三色区数值对齐 + 红色化（hotfix-v）
- 旧：绿色区画到 0.85，但 `_checkEscape` 实际安全区只到 0.70 → 玩家以为安全已开始跑鱼
- 新：直接从 `CONFIG.playing.tensionLowThreshold/HighThreshold` 读，永不脱钩
- 配色：[0, 40%) 红 + [40%, 70%] 绿 + (70%, 100%] 红，不透明，1px 米色边界线

### 3.6 钓获弹窗大鱼用 PNG（hotfix-v）
原是程序几何（橄榄绿椭圆+三角），改用 `getFishSpriteBySpecies` PNG（330px 宽等比），与水下/图鉴/鱼篓统一口径。
