## 二、鼠标左键全面与空格键并行（核心改动）

接到 Nina 反馈"水下场景拉回鱼也要鼠标左键"后，**横向拉通了所有"等待玩家敲空格"的入口**，用户单手用鼠标也能玩通整个钓鱼循环。

| 状态 | 旧操作 | 新操作 |
|---|---|---|
| Idle 抛竿蓄力 | 空格长按 | 空格 OR 鼠标左键长按 ✅（PHASE 21-1 D4 已做） |
| Playing 拉鱼 | 空格按住 | **空格 OR 鼠标左键按住** |
| BiteWindow 提竿 | 空格单击 | **空格 OR 鼠标左键单击** |
| Caught 再次抛竿 | 空格单击 | **空格 OR 鼠标左键单击** |
| Failed 跑鱼确认 | 空格单击 | **空格 OR 鼠标左键单击** |
| 钓鱼教程关闭 | 空格 | **空格 OR 鼠标左键单击按钮** |

**技术设计**：
- Playing"按住"独立状态字段 `_playingMouseHeld`（mousedown 在 Playing 时置位，mouseup 挂 window 防"虚按"，mouseleave 也清零）
- BiteWindow/Caught/Failed"单击"独立 flag `_confirmClick`（仅在三态下 mousedown 置位，update 消费后清零）
- 与 Aiming 蓄力鼠标流完全解耦（各自有 `fsm.is(...)` 守卫），不会互相污染
- **Bug fix**：`_renderPlaying` 内的 `holdingSpace` 一开始漏改了（鱼朝向只看键盘），第二次发版补上，现在鼠标拉鱼鱼会正确转向

**配套文案修订**（共 7 处）：
- BiteWindow 大字：`快按 [空格]或[鼠标左键] 提竿!`（绿色高亮整段）
- Caught 面板：`按 [空格] 或 [鼠标左键] 再次抛竿`
- Failed 红框：`请按 [空格]或[鼠标左键] 确认`（绿色高亮）
- Playing 木牌底栏第三行：`按住 [空格] 或 [鼠标左键] 控制拉力在绿色范围`（删除了"方向键 QTE 攻击鱼"——QTE 实际未启用）
- Playing 木牌主区：空格键帽右侧 +8px 新增**像素鼠标左键图标**（米白圆角矩形 + 中线分隔 + 左键霓虹绿高亮 + 顶部黄色"咔哒"光斑）
- 钓鱼技巧弹窗第一行：`按住[空格键]或[鼠标左键]和鱼搏斗，需要控制鱼线拉力，别断线哦！`
- 钓鱼技巧弹窗按钮：`按 [空格键] 或 [鼠标左键] 继续`（按钮也加了 cursor:pointer + mousedown 关闭）

**钓鱼技巧弹窗精简**：删了"按住 空格键 收线降低拉力，松开让拉力回升 / 保持拉力在 绿色区域 最容易成功！"整段（与第一行重复，按 Nina 要求剔除）。

## 三、村庄 · 阿明 sprite 锚点重构

**起因**：Nina 截图反馈"鼠标点击格子 vs 阿明位置错位"。

**根因**：PHASE-20 接入 sheet 时为兼容 legacy 32×48 像素小人，`drawAming` 把 `player.px/py` 当作 sprite 左上角而非格中心 —— 导致 sprite 视觉中心比逻辑 hitbox 中心**右偏 16px、上偏 10px**。

**方案**：仅改阿明 sprite，让"sprite 脚底踩在玩家逻辑中心 (player.px, player.py + T*3/8)"。

**最终参数**（经 3 次微调）：
- `drawAming(ctx, player.px - 16, player.py - 32, ...)` （初始 -24 → 上移 15px → 下移 7px = -32）
- `drawNameTag` 同步 `(player.px, player.py - 50)`
- ArrowGuide 头顶箭头 `arrowY = player.py - 53`

**未动**：所有 NPC 的渲染、阿明的碰撞 hitbox、寻路系统 —— 改动外科级、零回归。

## 四、村庄 · 昼夜系统重构

**Nina 决策**：12 分钟循环 + 4 阶段平均 3 分钟 + 滤镜浓度 ×0.5。

| 改动项 | 旧值 | 新值 |
|---|---|---|
| 完整循环 | 120 秒 | **720 秒（12 分钟）** |
| 阶段切分 | 0/0.15/0.50/0.70 不均匀 | **0/0.25/0.50/0.75 均匀** |
| 飘字数量 | 14 句细分 | **4 句**（黎明破晓 / 日正当中 / 黄昏降临 / 漫天星光） |
| 黎明粉滤镜 | 峰值 0.15 | **0.075** |
| 黄昏橙滤镜 | 峰值 0.12 | **0.06** |
| 夜晚冷蓝滤镜 | 封顶 0.32 | **0.16** |

**辅助同步**：太阳/月亮/星星/云朵的出现阈值全部对齐新边界；`getDarkness` 曲线重写但峰值（夜深 0.70）不变。

**外部联动**：`dusk-effects.js / reflections.js / buildings.js` 都用 `getDayNightPhaseName`（dawn/day/dusk/night），随新边界自动跟随，不需要单独改。

## 五、钓鱼浮漂全面升级（Nina 重点）

经过 6 轮迭代，从 8×8 圆形小方块进化成参考实物的"电子发光锥形浮漂"：

| 迭代 | 改动 |
|---|---|
| ① 重绘造型 | 7×22 栅格：发光头 + 红锥 + 黑环 + 黄环 + 黑环 + 白长身 + 黑尾杆 |

| ④ Idle 挂竿尖 | Idle/Aiming 时浮漂改绘到竿尖正下方 22px，鱼线由弧线变直线（"短鱼线垂挂"），不再悬在水面 |
| ⑤ 落水半隐 | `_drawPixelBob` 加 `clipBottom` 参数，水线下 12 行（白长身+黑尾）被裁掉 |
| ⑥ 围绕涟漪 | 新增 `_drawBobRipples`：3 层错峰扩散椭圆涟漪（俯视压扁 0.45），1.4s 周期，半径 11→37 |
