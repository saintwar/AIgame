# 🎣 《宝岛钓手》钓鱼核心玩法原型

> 单文件 HTML5 游戏，双击浏览器打开即可玩

## 🎯 游戏特性

- **5阶段状态机**：待机 → 瞄准蓄力 → 抛竿 → 等待 → QTE收线
- **2个核心节奏循环**：蓄力抛竿 + QTE方向键
- **4类失败分支**：过力断线、超时跑鱼、断线、挣脱
- **5种稀有度鱼**：从普通吴郭鱼到传说日月潭鱼王

## 🕹️ 操作说明

| 操作 | 键盘 |
|------|------|
| 抛竿/提竿/确认 | `E` 或 `空格` |
| 取消抛竿 | `Q` 或 `Esc` |
| QTE方向 | `↑↓←→` 或 `WASD` |
| 打开图鉴 | `B` |

## 📊 蓄力条说明

- 0-33%：灰色（弱力，落点近）
- 34-66%：绿色（中力，落点中）
- 67-95%：黄色（强力，落点远）
- 96-100%：红色（过力断线！）

## ⚙️ 调参指南

修改文件顶部的 `CONFIG` 对象即可调整手感：

```javascript
// 蓄力周期（秒）
chargeCycle: 1.5,

// 咬钩窗口（秒）
biteWindowDuration: 0.5,

// QTE相关参数
reeling: {
  qteCountByRarity: [3, 5, 7, 9, 11],      // 每个稀有度的QTE次数
  tickDurationByRarity: [0.8, 0.7, 0.6, 0.5, 0.4],  // 每拍限时
  hpDamageByRarity: [35, 20, 15, 12, 10],  // 每次成功扣鱼HP
  tensionPenaltyByRarity: [15, 18, 20, 22, 25],  // 每次失败加张力
}
```
